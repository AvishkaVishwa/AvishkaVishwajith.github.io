Self-Balancing Robot

Two-wheel self-balancing robot using IMU feedback and PID control for stable motion.

RoboticsIMUPIDControl

Overview

A two-wheel robot that balances upright using IMU feedback and real-time control loops.

Goals

  • Stable balance with smooth recovery
  • Responsive steering and speed control
  • Repeatable tuning for different loads

Hardware

  • Microcontroller with fast control loop
  • IMU for angle estimation
  • Motor driver and geared motors

Control Strategy

  • Sensor fusion for tilt estimation
  • PID loop for balance correction
  • Speed and turn commands layered on top
